#Example; Using the concepts to Check if a number is Odd or Even.
```
input_number = int(input("Enter a whole number: "))  
  
mod_input_number = input_number % 2  
  
if mod_input_number == 0:  
    print("even")  
else:  
    print("odd")
```

#### Nested if/else statements
```
input_number = int(input("Enter a whole number: "))
```
